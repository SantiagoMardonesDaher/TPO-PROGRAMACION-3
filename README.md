# TPO-PROGRAMACION-3
En este trabajo práctico abordamos el diseño, análisis e implementación de algoritmos aplicando tres paradigmas fundamentales estudiados durante la cursada: Divide y Vencerás, Greedy y Programación Dinámica.

Somos estudiantes de tercer año de la Universidad Argentina de la Empresa (UADE), y este proyecto forma parte de nuestra formación en el área de Algoritmos y Programación.


El proyecto esta hecho por Santiago Mardones Daher y Naim Gauna.
